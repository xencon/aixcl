AIXCL - AI & Infrastructure Stack Control CLI
=============================================

Unified command-line tool for managing AI services, monitoring stack, dashboards,
and models.


FEATURES
--------
- Start/stop/restart the entire stack
- Control individual services
- Manage LLM models and council configuration
- Launch dashboards (Grafana, OpenWebUI, pgAdmin)
- Environment validation & setup helpers
- Bash completion support
- Consistent, Docker-native workflows


INSTALLATION
------------
Clone the repository:

    git clone https://github.com/yourname/aixcl.git
    cd aixcl

Make the CLI executable:

    chmod +x aixcl.sh
    sudo ln -s $PWD/aixcl.sh /usr/local/bin/aixcl

(Optional) Install bash completion:

    aixcl utils bash-completion


USAGE
-----
    aixcl <command> <subcommand> [options]

View help for a specific command:

    aixcl help <command>


COMMAND OVERVIEW
----------------

STACK MANAGEMENT
    aixcl stack <start|stop|restart|status|logs|clean>

    start                Start all services
    stop                 Stop all services
    restart              Restart the entire stack
    status               Show service status
    logs [svc] [n]       Show logs for all or one service, optional line count
    clean                Remove unused Docker resources


SERVICE CONTROL
    aixcl service <start|stop|restart> <name>

Supported services:
    ollama, open-webui, postgres, pgadmin, watchtower,
    prometheus, grafana, cadvisor, node-exporter,
    postgres-exporter, nvidia-gpu-exporter, loki, promtail


MODEL MANAGEMENT
    aixcl models <add|remove|list> [name]

    add                  Add a model
    remove               Remove a model
    list                 List installed models


LLM COUNCIL
    aixcl council <configure|list>

    configure            Configure council behavior and members
    status               Show model status


DASHBOARDS
    aixcl dashboard <grafana|openwebui|pgadmin>


UTILITIES
    aixcl utils <check-env|bash-completion>

    check-env            Validate environment setup
    bash-completion      Install bash completion support


EXAMPLES
--------
Start the entire stack:

    aixcl stack start

Restart a single service:

    aixcl service restart postgres

Add a new LLM model:

    aixcl models add phi3

Open Grafana:

    aixcl dashboard grafana

Check environment requirements:

    aixcl utils check-env


BASH COMPLETION
---------------
Install:

    aixcl utils bash-completion

Or enable manually:

    source completion/aixcl.bash
